Setup — dostęp przez domenę (Let's Encrypt)

{{if not .HasDomain}}
Brak skonfigurowanej domeny. Uruchom lbpanel z flagą --domain lbpanel.twojadomena.pl lub ustaw zmienną środowiskową LBPANEL_DOMAIN=lbpanel.twojadomena.pl w pliku /opt/lbpanel/lbpanel.env i zrestartuj: systemctl restart lbpanel
{{end}}

Tryb 1 — dostęp przez IP (self-signed, działa od razu)

https://<IP_SERWERA>:4040

Przeglądarka pokaże ostrzeżenie o certyfikacie — kliknij "Zaawansowane → Przejdź mimo to".

{{if .HasDomain}}

Tryb 2 — domena {{.Domain}} z Let's Encrypt

Wklej do /etc/caddy/Caddyfile, następnie: systemctl reload caddy 

# /etc/caddy/Caddyfile

{
    admin 127.0.0.1:2019
    email admin@{{.Domain}}
}

www.{{.Domain}} {
    redir https://{{.Domain}}{uri} permanent
}

{{.Domain}} {
    reverse_proxy https://127.0.0.1:4040 {
        transport http {
            tls_insecure_skip_verify
        }
    }
    log {
        output file /var/log/caddy/lbpanel-access.log {
            roll_size 10mb
            roll_keep 5
        }
    }
}

DNS — ustaw w Cloudflare / rejestratorze

TypNazwaWartośćProxy
A {{.Domain}} <IP_SERWERA> ⚠ OFF (szara chmurka)
CNAME www.{{.Domain}} {{.Domain}} ⚠ OFF

Po uzyskaniu certyfikatu możesz włączyć Cloudflare proxy i ustawić SSL → Full (strict).
